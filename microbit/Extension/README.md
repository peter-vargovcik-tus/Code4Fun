# micro:bit extensions (deployment)

Published MakeCode extension packages for students.

| Extension | Folder | Import file |
|-----------|--------|-------------|
| **Code4Fun** | [Code4Fun/](Code4Fun/) | [SHARE_URL.txt](Code4Fun/SHARE_URL.txt) (Extensions search) |

## Workflow

1. Develop in `microbit/Gate/`.
2. Run `.\deploy-extension.ps1` from `microbit/`.
3. Give students the **Extensions URL** from `Extension/Code4Fun/SHARE_URL.txt`.

Full instructions: [Code4Fun/IMPORT.md](Code4Fun/IMPORT.md)
